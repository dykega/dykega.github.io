When you are developing a web page, it is easy to run a python program as a simple web server. Make sure your working directory is the directory that has your files in it, and then run python3 -m http.server This will start a webserver on port 8000. To bring up this page in your browser (running on the same machine) try this: http://localhost:8000/first.html
